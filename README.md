# proyecto_final_organizacion_empresarial_MELI
Proyecto segundo parcial - Organización empresarial (DaVinci)
Este repositorio contiene el proyecto final de la materia Organización Empresarial impartida por el profesor Facundo Ivan Romero de la carrera de Analista de Sistemas de Da Vinci.
El trabajo consiste en el desarrollo de una página web estática que simula el funcionamiento de una IA dentro del sitio de e-commerce MercadoLibre. El objetivo principal de esta IA es ayudar al cliente reuniendo diferente información sobre el producto de interés, como puede ser las características generales, pros y contras, métricas y finalmente preguntas frecuentes. 
El sitio se ha diseñado enteramente utilizando tecnologías como HTML, CSS y BOOSTRAP, simulando el sitio real de mercadolibre, con acceso a diferentes artículos, los cuales permiten ver la implantación de la IA en los artículos.
